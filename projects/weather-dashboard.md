## Overview

A weather dashboard that displays real-time weather data with interactive charts and maps. Integrates with OpenWeatherMap API for forecasts and historical data.

## Key Features

- Current weather conditions with icons
- 7-day forecast with charts
- Interactive weather maps
- Location-based search and auto-detect
- Severe weather alerts
- Historical data comparisons

## Tech Stack

- **Frontend:** React, Chart.js, Leaflet
- **API:** OpenWeatherMap
- **Styling:** Tailwind CSS
- **Deployment:** Vercel
