## Overview

This portfolio website built with Deno and Fresh framework. Features a clean, responsive design with dark/light mode, a markdown-based blog with syntax highlighting, and a project showcase.

## Key Features

- Dark/light mode with persistent preference
- Markdown-based blog with syntax highlighting
- Project showcase with markdown content
- Responsive design across all devices
- Scroll-triggered animations
- Accessibility-first approach
- Zero client-side JS overhead (islands architecture)

## Tech Stack

- **Runtime:** Deno
- **Framework:** Fresh
- **UI:** Preact + Tailwind CSS
- **Blog:** Markdown + marked + highlight.js
- **Deployment:** Deno Deploy
