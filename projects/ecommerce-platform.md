## Overview

A full-stack e-commerce platform built with React and Node.js. Features include product catalog with search and filtering, shopping cart, Stripe payment integration, user authentication, and an admin dashboard for inventory management.

## Key Features

- Product catalog with search, filter, and pagination
- Shopping cart with persistent state
- Stripe payment integration
- User authentication and profile management
- Admin dashboard for inventory and order management
- Responsive design for mobile and desktop

## Tech Stack

- **Frontend:** React, Tailwind CSS, Redux
- **Backend:** Node.js, Express
- **Database:** MongoDB
- **Payments:** Stripe API
- **Deployment:** Vercel + Railway
